---
title: "Edition volunteers"
description: "Manage volunteers assigned to an edition"
---

The volunteers endpoint returns all volunteers assigned to a given edition, grouped by their role type: **Referee**, **Judge**, and **Floater**.

<Info>
  Authentication is required to access volunteer data.
</Info>

---

## List volunteers for an edition

```
GET /editions/{editionId}/volunteers
```

Returns the volunteers assigned to the edition, partitioned into three lists by role.

**Authentication required.**

### Path parameters

<ParamField path="editionId" type="integer" required>
  The numeric ID of the edition.
</ParamField>

### Errors

| Error code | HTTP status | Cause |
|---|---|---|
| `EDITION_NOT_FOUND` | `404` | No edition exists with the given `editionId`. |

<Note>
  If the edition exists but has no volunteers of a given type, the corresponding list is empty (`[]`), not absent.
</Note>
